Model Context Protocol Server

Build AI-powered MCP tools from your JavaScript APIs with ease

⚠️ This is an MCP (Model Context Protocol) server, not a web application. Use MCP clients to connect and interact with the available tools.

🚀 What is Easy MCP Server?

Easy MCP Server is a powerful framework that automatically generates MCP tools from your JavaScript API classes. Simply create files in the api/ directory where the file path represents the API endpoint and the filename represents the HTTP method (get.js, post.js, etc.). It creates both REST endpoints and MCP tools that AI models can understand and use.

Key Features:

✨ Automatic OpenAPI documentation generation
🔄 Real-time hot reloading during development
🌐 Multiple transport protocols (HTTP, WebSocket, SSE)
🛠️ Built-in MCP tool discovery and execution

🔗 Available Endpoints

GET / - This information page
POST /mcp - MCP protocol endpoint
GET /sse - Server-Sent Events for real-time updates

🤖 How to Connect

1. Using MCP Inspector:

npx @modelcontextprotocol/inspector

Then use URL: http://localhost:3001

2. Using Claude Desktop:

Add this server to your Claude Desktop configuration:

http://localhost:3001

3. Using Cursor IDE:

Add to your Cursor MCP configuration file:

~/.cursor/mcp.json

Configuration:

{ "mcpServers": { "easynet": { "url": "https://mcp.easynet.world/", "transport": "http" } } }

4. Try our live demo:

Experience Easy MCP Server in action at:

https://mcp.easynet.world

5. Using other MCP clients:

Connect to this server using the MCP protocol over HTTP.

🛠️ Available MCP Commands

tools/list - Discover available API endpoints

tools/call - Execute a specific API endpoint

ping - Health check

📦 Installation & Usage

Install via NPM:

npm install easy-mcp-server

Quick Start:

npx easy-mcp-server

How it works:

Create API files in the api/ directory:

api/users/get.js → GET /users api/users/post.js → POST /users api/orders/put.js → PUT /orders

Or use our live demo:

Visit https://mcp.easynet.world to see Easy MCP Server in action!
